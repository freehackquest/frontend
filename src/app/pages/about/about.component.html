Distribution

Install fhq-server from repository

Ubuntu repo: https://launchpad.net/~freehackquest/+archive/ubuntu/fhq-server/+packages 
$ sudo add-apt-repository ppa:freehackquest/fhq-server
$ sudo apt-get update
$ sudo apt install fhq-server

Install fhq-server from docker

Docker hub: https://hub.docker.com/r/freehackquest/fhq-server

Docker-compose example: docker-compose.yml

$ docker pull freehackquest/fhq-server

Source code

frontend (Required fhq-server): http://github.com/freehackquest/frontend

fhq-server: http://github.com/freehackquest/fhq-server
